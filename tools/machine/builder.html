Frame

Machine length along its long axis. 5 = compact, 14 = full-size reference.
Machine thickness front-to-back. 2 = narrow, 3.6 = reference depth.

Section Ratio

Resource + counter area. 0.4 = 40% of width. Bigger = more room for essence display.
Gear animation area. 0.2 = 20% of width. Smaller = compact gears, bigger = spread out.
Product display area. 0.4 = 40%. All three must sum to 1.0.

Scale Overrides

Gear disk + teeth radius. 1.0 = auto, 1.2 = 20% bigger gears.
Rotation speed. 0 = frozen, 1 = normal, 3+ = fast.
Essence/resource mesh on the input section. 1.5 = 50% bigger.
Candy/product mesh on the output section. 1.5 = 50% bigger.
The "0/10" text billboard above the input. 1 = auto, 2 = double.
Corner pillars of the frame. 1 = auto, 2 = twice as tall.
Green "stand here" drop-off zone in front of inputs. 1 = auto.
Y-axis rotation for input + output meshes. 0 = flat, 1.57 = 90° perpendicular, 3.14 = flipped.
