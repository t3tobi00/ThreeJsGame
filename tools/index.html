Dev Tools

UI utilities for editing game config without touching JSON by hand.

🧍 Player Editor

Tweak the player archetype — speed, stack spacing, accepted resources, health, animation curves, and more. Outputs the full player.json ready to paste back.

🟦 Unlock Zone Builder

Build unlock-zone JSON for level files — build/spawner/convert types, costs, and output targets. Outputs a single zone entry to paste into level-1.json.

⚙ Machine Builder

Configure gearworks machines — select input resources, amounts, output type, and frame dimensions. Live 3D preview with copyable JSON for level-1-diorama.json.

👾 Enemy Editor

Tune the enemy archetype (speed, HP, AI, damage, drops), global spawn config (interval, fixed spawn point, cap, despawn distance), and per-level counts. Three copy-paste outputs.

Run a local web server from the project root (e.g. python3 -m http.server) then open tools/index.html in your browser. The tools fetch live config from src/config/, so any new archetypes/resources show up automatically.
